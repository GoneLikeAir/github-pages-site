GitHub Pages Starter

一个可以直接托管到 GitHub Pages 的静态网页项目

这是一个零构建、零后端的 HTML/CSS/JS 项目。推到 GitHub 后，开启 Pages 即可在线访问。

查看 GitHub Pages 部署步骤
⚡

无需构建

直接由浏览器加载 index.html、styles.css 和 script.js。

🌐

适合 Pages

仓库推送后，可以用 GitHub Pages 的 branch source 或 Actions 部署。

🧩

易于扩展

可以替换成 HTML Anything 导出的页面，也可以继续添加更多页面和资源。

部署到 GitHub Pages

  1. 在 GitHub 创建一个公开仓库，比如 github-pages-site。
  2. 把本项目 push 到仓库的 main 分支。
  3. 进入仓库 Settings → Pages。
  4. 选择 Deploy from a branch，分支选 main，目录选 /root。

如果仓库名是 GoneLikeAir.github.io，访问地址是 https://GoneLikeAir.github.io；普通项目仓库则是 https://GoneLikeAir.github.io/仓库名/。
