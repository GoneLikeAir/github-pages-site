# AI Image Grimoire

翻页式 AI 生图 / 美术提示词教程网页。

## 打开方式

```bash
cd /Users/gone/AI_Image_Grimoire
python3 -m http.server 8765
# 浏览器打开 http://127.0.0.1:8765/
```

## 内容

- 12 个连页章节：提示词结构、构图、光线、色彩、镜头、材质、电影摄影、水墨水彩、3D 潮玩、海报、信息图、角色设定表。
- 每页包含：核心概念、可复用公式、关键词、常见误区、示例咒语、练习和 GPT-Image 示例图。
- 支持：目录搜索、缩略图跳转、键盘翻页、移动端响应式、reduced-motion。
